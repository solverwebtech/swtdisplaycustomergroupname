# Display Customer Group (swtdisplaycustomergroupname)

## Description
This PrestaShop module displays the logged-in customer's group name with a profile icon.  
It can also be rendered using a custom hook.

## Features
- Displays customer group name
- Supports custom hook
- Lightweight & override safe
- Compatible with PrestaShop 8.x
